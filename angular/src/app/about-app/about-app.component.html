Description:

This is a dynamic single-page-application. A Web App which creates a shoppingList, in which u can add your preferred listname, itemname and its quantity.

Features:

This app featues the CRUD operations (Create, Read, Update, Delete). The lists which you see on the table is from a database, created in MongoDB. The database is read in a scrollable mat table. The creation and the editing of a list are in dialogs.

Use of Technologies and Languages:

  • Mongo DB
  • Express
  • Angular JS
  • Node.js
  • HTML5
  • CSS3
  • JS
  • Bootstrap
  • TypeScript
